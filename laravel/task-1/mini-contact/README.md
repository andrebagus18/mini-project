# 🚀 Laravel Starter

A reusable Laravel 13 starter template for rapid development.

## ✨ Features

- Laravel 13
- Tailwind CSS
- SweetAlert2
- Vite
- Clean Project Structure

## 📦 Requirements

- PHP 8.3+
- Composer
- Node.js 20+

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/andrebagus18/laravel-starter.git
```

Go to project

```bash
cd laravel-starter
```

Install PHP dependencies

```bash
composer install
```

Install Node dependencies

```bash
npm install
```

Copy environment

```bash
cp .env.example .env
```

Generate application key

```bash
php artisan key:generate
```

Configure your PostgreSQL database in the `.env` file.

Run migration

```bash
php artisan migrate
```

Start development server

```bash
php artisan serve
```

Run Vite

```bash
npm run dev
```
